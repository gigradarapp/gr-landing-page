# Buzo Landing Page

Public company thesis page for Buzo / GigRadar.

This site is intentionally separate from:

- `gr-frontend`: the usable Buzo app
- `gr-waitlist`: the acquisition / waitlist page

Its job is to explain the company thesis: Buzo is building the decision layer for going out, starting with Singapore nightlife.

## Stack

- Astro
- TypeScript
- Custom CSS
- Static Vercel deployment

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```
